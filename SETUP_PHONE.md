# SETUP_PHONE.md — Your ONE HOUR with Ammamma's phone

Do these IN ORDER on the Oppo (ColorOS 5.2). Tick each one. ~30 min total,
leaving time to sit with her and show her the app.

## A. Install (5 min)
1. Send the APK to the phone (Telegram → Saved Messages → tap the APK).
2. If blocked: Settings → Additional Settings → Safety & Privacy →
   allow "Install from unknown sources" for Telegram. Install.
3. Open the app once. **Accept EVERY permission dialog it shows** (calls,
   contacts, SMS, location, microphone). It asks only this one time.
4. When it opens the "ignore battery optimizations" dialog → **Allow**.
5. When it opens "Display over other apps" → find **Ammamma tōḍu** → turn **ON**.
   (This is what lets alert screens actually appear. Do not skip.)

## B. Make ColorOS never kill it (10 min — THE most important part)
6. Settings → App Management (or Security app) → **Startup Manager** /
   Auto-start → allow **Ammamma tōḍu**.
7. Settings → Battery → **Energy Saver** → Ammamma tōḍu → turn OFF
   "Background freeze" and "Automatically optimize when an abnormality is detected".
8. Open Recents (square button) → find the app card → **pull it down** (or tap
   the lock icon) so a padlock shows. Locked = "clean all" won't kill it.
9. Settings → App Management → Ammamma tōḍu → Permissions → check everything
   is allowed (ColorOS sometimes silently revokes).

## C. Voice quality (5 min)
10. Settings → Additional Settings → Language & Input → Text-to-speech output →
    engine = **Google Text-to-speech**. Tap its ⚙ → Install voice data →
    **Telugu** (needs Wi-Fi once; then works offline).
11. In the same screen, slow the Speech rate one notch below normal —
    clearer for her.

## D. Phone settings (3 min)
12. Turn **Location ON** (the app will nag with a red banner if it's off — good).
13. Ring volume to maximum. Media volume the app manages itself.
14. Make sure her SIM has SMS balance (find-my-phone replies by SMS).

## E. In the app (5 min)
15. Long-press the gear (top-right) → Settings: set the **code word**, add
    **family numbers**, paste the **OpenRouter key** (for Talk).
16. Add/verify her people on the home grid (long-press a card to edit).

## F. TEST — do not leave without all four passing (10 min)
17. **Call her phone** from yours → it must SPEAK your name over the ringtone.
18. **Text the code word** → loud alarm + a screen with a big STOP button.
    If no screen appears: recheck step 5, then step 6.
19. **Unplug → plug the charger** → charging screen appears + speaks.
20. In Settings, press every **demo button** once so you hear each alert.
21. Final: reboot the phone, wait 2 min, call it again → it must still speak.
    That proves the boot-restart works. Now go sit with her. ❤️
